# CopyrightPass (CRP)

CopyrightPass (CRP) is a digital tool token designed for IP licensing, PLR/PPR rights verification, DAO governance, and Merkle Tree asset records.

CRP supports a verifiable IP authorization framework where certified IP assets can be connected to on-chain records, governance processes, and production-right verification.
